# PciHi Messenger

[![Лицензия](https://img.shields.io/badge/license-MIT-blue.svg)](LICENSE.md)  
[![Версия](https://img.shields.io/badge/version-1.0.0-green.svg)]()

Простой, безопасный и элегантный мессенджер для повседневного общения.

## О проекте

PciHi — это современный мессенджер, созданный для комфортного обмена сообщениями в реальном времени. Мы сделали ставку на:
* **Минималистичный интерфейс** — ничего лишнего, только общение.
* **Безопасность** — сквозное шифрование всех переписок.
* **Скорость** — мгновенная доставка сообщений даже при слабом соединении.
* **Кросс‑платформенность** — работает на Linux (Arch точно) и Windows 7/8/10/11.

## Возможности

* Обмен текстовыми сообщениями
* Отправка изображений и файлов
* Групповые чаты (до 100 участников)
* Сквозное шифрование (end‑to‑end)
* Уведомления в реальном времени
* История переписки с поиском
* Поддержка эмодзи и форматирования текста
* Тёмная/светлая тема интерфейса

## Установка

### Windows
1. Скачайте установщик `PciHi-Setup-v1.0.0.exe` из раздела Releases.
2. Запустите файл и следуйте инструкциям мастера установки.
3. После установки запустите PciHi из меню «Пуск» или с рабочего стола.

### Linux
1. Скачайте пакет `pcihi_1.0.0_amd64.deb` (для Debian/Ubuntu) или `pcihi-1.0.0-1.x86_64.rpm` (для Fedora/CentOS).
2. Установите через терминал:
   ```bash
   sudo dpkg -i pcihi_1.0.0_amd64.deb  # для Debian/Ubuntu
   ```
   или
   ```bash
   sudo rpm -i pcihi-1.0.0-1.x86_64.rpm  # для Fedora/CentOS
   ```
3. Запустите приложение из меню приложений.

## Начало работы

1. Запустите PciHi.
2. Зарегистрируйтесь, указав email и пароль.
3. Подтвердите email через письмо в почте.
4. Добавьте контакты через поиск по email или пригласите друзей по ссылке.
5. Начните общение!

## Настройки

В разделе «Настройки» вы можете:
* Изменить тему интерфейса (светлая/тёмная).
* Настроить уведомления.
* Управлять конфиденциальностью.
* Изменить пароль и email.
* Экспортировать историю переписки.

## Разработка

Хотите внести вклад в проект? Ознакомьтесь с [GUIDE_FOR_CONTRIBUTORS.md](GUIDE_FOR_CONTRIBUTORS.md).

### Сборка из исходников

1. Клонируйте репозиторий:
   ```bash
   git clone https://github.com/pcichi/messenger.git
   ```
2. Установите зависимости:
   ```bash
   npm install
   ```
3. Соберите проект:
   ```bash
   npm run build
   ```
4. Запустите:
   ```bash
   npm start
   ```
